JSON (Javascript Object notation)
====
JSON is the universal format 
JSON is used for the storing and exchanging data
JSON store the data in Text
JSON follow key values structure
any decimal and number does not reuired ""
structure
------------
{
  "Key" : "Value"
}
e.g
{
  "Name" : "Yogesh",
  "Phone" : 1234
}

create student data in JSON with first name and last name ,Phone
{
   "FirstName": "YOGESH",
   "LastName": "Pokharkar" ,
   "Phone" : "8421099"
}

create the json to store array of the city
{
  "City" : ["hyd", "Chen","bglr"]
}

create a json for storing the account information
{
    "Account" : [
        {"Name" : "EPAM sys" ,"Industry" : "Education" "rating" : "Hot" , "Phone" : "123"},
        {"Name" : "Ganesh sys" ,"Industry" : "Education" "rating" : "Hot" , "Phone" : "123"},
        {"Name" : "ABC sys" ,"Industry" : "" "rating" : "Hot" , "Phone" : "123"}
    ]} 

create json to store ages
{
   "Age" :[10,20,30,40]
}


serilization
=============
conveting an object data into json is call serilization.

Classes in salesforce to serilization and Deseralize
-----------------------------------------------
System namespace
JSON Class - Manual Serilize / Deseralize
JSONGenerator Class - Automatic/Dynamic - serilization
JSONParser Class - Automatic/Dynamic - serilization
JSONToken Enum - for genrating the enumarated token

deserilization
==============
converting JSON data Back to a Object form is called as deserilization.

deserialize(JsonString, ApexType) => deserializes the specified JSON String into an apex object of the specified type.
deserializeStrict(JsonString, ApexType) => deserializes the specified JSON String into an apex object of the specified type.
deserializeUntyped(JsonString) => deserializes the specified JSON String into collection of primitive datatype.
